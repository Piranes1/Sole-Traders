# Sole Traders Web Application

This project was developed as part of the **CSC7084 Web Development** module on the **MSc Software Development** programme at **Queen’s University Belfast**.

The application provides a space where local traders can create accounts, list the services they offer, and manage their availability. Users can browse traders, view their services, and contact them to arrange bookings.

## Features

- Trader account creation and authentication  
- Service listings with descriptions and pricing  
- User browsing and search functionality  
- Booking and contact workflow  
- Ratings and feedback system
- Interactive charts for visualising trader performance and activity  
- Backend API and SQL database integration  

## Technologies Used

- Node.js / Express  
- SQL database  
- HTML, CSS, JavaScript  
- RESTful API architecture  

## Purpose

The aim of this project was to design and implement a full‑stack web application demonstrating modern web development practices, including API design, database modelling, and user‑focused interface development.
